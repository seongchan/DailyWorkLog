    핵심 디자인 원칙

    1 파스텔톤 컬러 — 저채도 소프트 파스텔로 눈의 피로도를 줄입니다.
    2 저채도 버튼 — 저장/취소/삭제 모두 무채색 계열. .mod-cta 사용 금지.
    3 미니멀 ✕ 삭제 버튼 — 평소 투명, hover 시만 빨간 텍스트. 배경색 변화 없음.
    4 팝업은 시스템 테마를 따름 변경 — 라이트/다크 테마에 따라 자동 전환 (CSS Canvas/CanvasText).
    5 Obsidian CSS 변수 우선 — --text-normal 등으로 테마 호환성 유지.
    6 다국어 지원 (EN / KO) 추가 — 기본값 English. 플러그인 설정에서 전환. 파서는 양 언어 마커를 항상 동시 인식.
